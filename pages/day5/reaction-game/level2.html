Catch a False Start

LEVEL 2 · ON YOUR OWN
Name:  

The challenge

Right now, pressing a button during "Get Ready..." does nothing, the game just ignores it and keeps waiting. Real reaction games punish that. If either player presses before the LED turns on, end the round immediately and declare the other player the winner, with the LCD showing something like "P1 false start! P2 wins."

Your task

During the "Get Ready" wait, keep checking both buttons

If a button is pressed during that wait, skip the LED flash entirely

Display a false start message naming the guilty player and the winner by default

Still support "play again"

Hint 1: sleep() can't check anything

Your current wait uses sleep(), which does nothing but pause, it can't detect a button press while it's running. Replace it with a loop that checks the elapsed time and the buttons on every pass instead.

Hint 2: tracking elapsed time inside a loop 
wait_time = random.uniform(2, 5)
start = time()
false_start = None

while time() - start < wait_time:
    if button1.is_pressed:
        false_start = "Player 1"
        break
    elif button2.is_pressed:
        false_start = "Player 2"
        break
Hint 3: full structure, if you're really stuck 
if false_start is not None:
    other = "Player 2" if false_start == "Player 1" else "Player 1"
    lcd.clear()
    lcd.write_string(false_start + " false start!")
    lcd.cursor_pos = (1, 0)
    lcd.write_string(other + " wins!")
else:
    led.on()
    lcd.clear()
    lcd.write_string('GO!')
    start_time = time()
    # ...rest of the normal game as before
