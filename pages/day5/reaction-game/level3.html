Track a Running Score

LEVEL 3 · HARDEST
Name:  

The challenge

Right now every round forgets the last one. Turn this into a real match: keep score across rounds, show the running score on the LCD after every round, and once a player reaches 3 wins, declare them the champion and stop the game.

Why this is genuinely hard

Every previous step reset itself each round. This one doesn't. You need a variable that survives from one round to the next, gets updated, not replaced, and gets checked against a win condition every single round. That's real state management, the same idea behind saving progress in a video game.

Your task

Create two counters before the game loop starts, one per player, both starting at 0

After each round, add 1 to the winner's counter

Show the current score on the LCD's second line, replacing the reaction time display

The moment a player's counter hits 3, show a champion message and stop asking "play again"

Hint 1: counters that live outside the loop

Declare p1_score = 0 and p2_score = 0 once, before your while again == "y" loop begins. If you declare them inside the loop instead, they'll reset to 0 every round, which defeats the whole point.

Hint 2: updating the right counter 
if winner == "Player 1":
    p1_score += 1
else:
    p2_score += 1
Hint 3: full structure, if you're really stuck 
p1_score = 0
p2_score = 0
champion = None

while champion is None:
    # ...run one full round, get `winner`...

    if winner == "Player 1":
        p1_score += 1
    else:
        p2_score += 1

    lcd.clear()
    lcd.write_string(winner + " wins!")
    lcd.cursor_pos = (1, 0)
    lcd.write_string(f"P1:{p1_score}  P2:{p2_score}")

    if p1_score == 3 or p2_score == 3:
        champion = "Player 1" if p1_score == 3 else "Player 2"
        lcd.clear()
        lcd.write_string(champion + " wins the match!")
