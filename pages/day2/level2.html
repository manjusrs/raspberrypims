Independent Challenge · Day 2

Build a Traffic Light Mode

LEVEL 2 · ON YOUR OWN
Name:  

The challenge

Sliders are fine for manual control, but a real traffic light runs itself. Add one button to your GUI, labeled "Traffic Light Mode," that automatically cycles LED 1 (red), LED 2 (yellow), and LED 3 (green) through a realistic sequence: red, then green, then yellow, then back to red.

Why sleep() will break your GUI

Your instinct might be to write a function with sleep() between each light change, the same way you did on Day 1. Try it. You'll notice the entire window freezes while it runs, no sliders respond, nothing.

That's because sleep() pauses your whole program, and Tkinter needs to keep running constantly in the background to stay responsive. Tkinter has its own way of waiting, without freezing: window.after(milliseconds, function_name).

Your task

Add a button to the GUI labeled "Traffic Light Mode"

Write a function for each stage of the sequence (red, green, yellow)

Use window.after() to schedule the next stage, instead of sleep()

Make the cycle repeat on its own once started

Hint 1: what after() actually does

window.after(2000, show_green) tells Tkinter "in 2000 milliseconds, run the function show_green," then immediately continues running everything else. It doesn't pause anything.

Hint 2: chaining stages together

Each stage function can schedule the next one at its own end. That's how you build a self-repeating cycle without a loop.

def show_red():
    led1.value = 1
    led2.value = 0
    led3.value = 0
    window.after(2000, show_green)
Hint 3: full structure, if you're really stuck 
def show_red():
    led1.value = 1; led2.value = 0; led3.value = 0
    window.after(2000, show_green)

def show_green():
    led1.value = 0; led2.value = 0; led3.value = 1
    window.after(2000, show_yellow)

def show_yellow():
    led1.value = 0; led2.value = 1; led3.value = 0
    window.after(500, show_red)

tk.Button(window, text="Traffic Light Mode", command=show_red).pack()

Push further (optional)

Add a second button that stops the cycle mid-sequence and hands control back to the manual sliders. What state do you need to track to make that work cleanly?
