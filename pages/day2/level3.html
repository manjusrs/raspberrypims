Independent Challenge · Day 2

Make the Buzzer Track the Lights

LEVEL 3 · HARDEST
Name:  

The challenge

Right now the buzzer has its own independent slider. Disconnect that idea completely. Instead, the buzzer's volume should update automatically, live, based on the average brightness of your three LEDs. Turn the LEDs up, the buzzer gets louder on its own, with no separate slider touched.

Why this is genuinely hard

Your LED sliders each call their own small function the moment they move. But the buzzer needs to react to all three at once, and none of those three functions know about each other. You need a shared piece of logic that all three sliders trigger, that recalculates the average and updates the buzzer every time.

Your task

Remove the buzzer's independent slider

Write one function that reads all three LED brightness values and sets the buzzer's volume to their average

Call that function from all three LED sliders, not just one

Make sure the master button still silences the buzzer along with the LEDs

Hint 1: one function, called from three places

Your slider functions currently only set one LED's value each. Add a second line at the end of each one that calls a new function, something like update_buzzer().

Hint 2: reading current slider values

PWMLED objects remember their own current value. You don't need to track brightness separately, you can just read led1.value, led2.value, and led3.value directly, any time.

Hint 3: full structure, if you're really stuck 
def update_buzzer():
    average = (led1.value + led2.value + led3.value) / 3
    buzzer.value = average

def set_led1(val):
    led1.value = int(val) / 100
    update_buzzer()

def set_led2(val):
    led2.value = int(val) / 100
    update_buzzer()

def set_led3(val):
    led3.value = int(val) / 100
    update_buzzer()

Push further (optional)

Right now every LED counts equally toward the average. What would you change if LED 1 should count twice as much as the other two? You don't need to build it, just describe the change to update_buzzer() in words.
