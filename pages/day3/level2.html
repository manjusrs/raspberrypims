Independent Challenge · Day 3

Trade the Light for a Sound

LEVEL 2 · ON YOUR OWN
Name:  

The challenge

Everything you built alerts you with light. Rebuild it so it alerts you with sound instead. When your hand gets within 15cm, the buzzer should turn on, exactly the same threshold logic, a different output.

Your task

Wire a buzzer to an open GPIO pin and GND (GPIO 27 is free)

Import the right class for the buzzer

Replace every place your code touches the LED with the buzzer instead

Keep the GUI label and toggle button working exactly as before

Hint 1: which class to use

You've used two different buzzer-related classes this week. For a simple on/off buzzer, plain Buzzer from gpiozero works, no PWM needed for this one.

Hint 2: find every LED reference

Search your code for every line with led in it. Each one needs a buzzer equivalent: the import, the object creation, the .on(), and the .off().

Hint 3: full structure, if you're really stuck 
from gpiozero import DistanceSensor, Buzzer

sensor = DistanceSensor(echo=24, trigger=23)
buzzer = Buzzer(27)

def check_sensor():
    if system_on:
        cm = sensor.distance * 100
        label.config(text=f"{cm:.1f} cm")
        if cm < 15:
            buzzer.on()
        else:
            buzzer.off()
    window.after(200, check_sensor)
