Build a Real Parking Sensor

LEVEL 3 · HARDEST
Name:  

The challenge

You've probably heard a real car's parking sensor: a slow beep when something's far away, speeding up into a solid tone as you get close. Build that. The buzzer should beep faster and faster as your hand approaches, not just switch on and off at one threshold.

Why this is genuinely hard

Level 2 only had two states: on or off. This challenge has a continuous range, the beep speed itself is a number that changes based on another number, the distance. You'll need a function that converts distance into a delay time, and a way to keep re-scheduling the next beep using that changing delay.

Your task

Write a function that converts a distance in cm into a beep delay in milliseconds, closer distance means a smaller delay

Use window.after() to schedule each beep, using that delay

Beyond a certain distance, stop beeping entirely, silence

Keep the live distance label and toggle button working

Hint 1: mapping one range to another

You need to turn "distance from 0 to 50cm" into "delay from 100ms to 800ms." That's the same kind of math you used for the Day 1 fading LED, just with different numbers. Closer distance should produce a smaller delay.

Hint 2: a beep that reschedules itself

A single beep function can turn the buzzer on briefly, then off, then call window.after() again with a freshly recalculated delay, based on whatever the distance is right now. That's how the speed changes over time without a while loop.

def distance_to_delay(cm):
    if cm > 50:
        return None
    return int(100 + (cm / 50) * 700)
Hint 3: full structure, if you're really stuck 
from gpiozero import DistanceSensor, Buzzer

sensor = DistanceSensor(echo=24, trigger=23)
buzzer = Buzzer(27)

def distance_to_delay(cm):
    if cm > 50:
        return None
    return int(100 + (cm / 50) * 700)

def beep_cycle():
    if not system_on:
        return
    cm = sensor.distance * 100
    label.config(text=f"{cm:.1f} cm")
    delay = distance_to_delay(cm)
    if delay is None:
        buzzer.off()
        window.after(200, beep_cycle)
    else:
        buzzer.on()
        window.after(50, buzzer.off)
        window.after(delay, beep_cycle)

beep_cycle()
