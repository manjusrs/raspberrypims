Merge the Sensor Into the Booth

LEVEL 3 · HARDEST
Name:  

The challenge

Combine everything from today. The booth should fire automatically when someone steps within range, exactly like this morning, but now it applies whichever filter is currently selected in the GUI, exactly like Level 2. The physical shutter button should still work too, as a manual override. The system LED should still reflect whether the booth is armed.

Why this is genuinely hard

This morning's sensor code ran inside a plain while True loop. This afternoon's GUI needs Tkinter's own loop, window.mainloop(), running instead. You can't run two different infinite loops at the same time in one script. You need the sensor check to happen inside Tkinter's loop, using the same window.after() pattern you've used all week, not a separate while True.

Your task

Remove any standalone while True sensor loop

Use window.after() to check the sensor repeatedly, on a timer, inside the GUI

When someone's close enough, auto-capture using the currently selected filter, same variable as Level 2

Keep a cooldown so it doesn't fire repeatedly for the same person

The system toggle button and LED still work, and pause the sensor checks while off

Hint 1: a self-rescheduling sensor check

Write one function that checks the sensor once, does its job if needed, and then calls window.after() on itself to check again a little later. This is the same shape as Day 3's check_sensor().

Hint 2: cooldown without sleep()

sleep() would freeze the GUI here too. Instead, track the time of the last capture in a variable, and compare it to the current time before allowing another capture.

from time import time

last_capture = 0

def check_sensor():
    global last_capture
    if system_on:
        cm = sensor.distance * 100
        if cm < 30 and time() - last_capture > 3:
            take_photo()
            last_capture = time()
    window.after(200, check_sensor)
Hint 3: full structure, if you're really stuck 
system_on = True

def toggle_system():
    global system_on
    system_on = not system_on
    led.on() if system_on else led.off()

toggle_button.when_pressed = toggle_system

def take_photo():
    filename = datetime.now().strftime("booth_%H%M%S.jpg")
    camera.capture_file(filename)
    img = Image.open(filename)
    filtered = current_filter(img)
    display_img = ImageTk.PhotoImage(filtered.resize((320, 240)))
    photo_label.config(image=display_img)
    photo_label.image = display_img

shutter.when_pressed = lambda: window.after(0, take_photo)

last_capture = 0

def check_sensor():
    global last_capture
    if system_on:
        cm = sensor.distance * 100
        if cm < 30 and time() - last_capture > 3:
            take_photo()

check_sensor()
