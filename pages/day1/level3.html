Build a Mode-Switching Light

LEVEL 3 · HARDEST
Name:  

The challenge

You've now got a button and a PWM-capable LED wired into the same breadboard. Combine them. Every time the button is pressed, the LED should switch to the next mode in a cycle of three:

ModeBehavior
1Off
2Solid on, full brightness
3Breathing fade, like your Level 2 code

Press the button once more from mode 3, and it should loop back to mode 1.

Why this is genuinely hard

Your fade code from Level 2 uses while True and sleep(), which blocks the program from checking the button at the same time. You need the LED to keep breathing while still watching for a button press. That's a real problem professional programmers run into constantly, it's called blocking code.

Your task

Keep a variable that tracks the current mode number

When the button is pressed, advance the mode and wrap back to 1 after 3

In your main loop, check the mode variable every cycle and act accordingly

Make sure the fade animation still responds to a button press mid-fade

Hint 1: don't use sleep() in the main loop

Instead of one big sleep(0.05) that freezes everything, try tracking brightness with a variable that increases a tiny bit each pass through a fast loop, and check the button every single pass.

Hint 2: when_pressed doesn't need to pause anything

Set button.when_pressed to a function that just changes the mode variable. It runs instantly and returns control right back to your main loop, it doesn't block anything.

mode = 0

def next_mode():
    global mode
    mode = (mode + 1) % 3

button.when_pressed = next_mode
Hint 3: full structure, if you're really stuck 
from gpiozero import PWMLED, Button
from time import sleep

led = PWMLED(17)
button = Button(2)
mode = 0

def next_mode():
    global mode
    mode = (mode + 1) % 3

button.when_pressed = next_mode

brightness = 0
direction = 1

while True:
    if mode == 0:
        led.value = 0
    elif mode == 1:
        led.value = 1
    elif mode == 2:
        brightness += direction * 0.02
        if brightness >= 1 or brightness <= 0:
            direction *= -1
        led.value = max(0, min(1, brightness))
    sleep(0.02)
