Make Your LED Breathe

LEVEL 2 · ON YOUR OWN
Name:  

The challenge

So far your LED has only known two states: on or off. Real dimmer switches, phone flashlights, and mood lighting can be anywhere in between. Your job: make your LED smoothly fade from off to fully bright, then back down, like it's breathing.

What you know already

You've used the LED class, which only understands on() and off(). That's not enough for this. You'll need a different class from the same library: PWMLED. It accepts a brightness value between 0 (off) and 1 (full brightness), including decimals like 0.3.

Your task

Swap your circuit's LED wiring to use the same GPIO 17 pin, no rewiring needed, this is a code-only challenge

Import PWMLED instead of LED

Write a loop that raises the brightness from 0 to 1 in small steps

Write a second loop that lowers it back from 1 to 0

Wrap both in an outer loop so it breathes continuously

Hint 1: how do I count in decimals?

range() only works with whole numbers. You'll want a whole number counter, then divide it by something to turn it into a decimal brightness value. Think: if you count from 0 to 10, how do you turn each step into a number between 0 and 1?

Hint 2: the shape of the loop

You need two directions: brightness going up, then brightness going down. That's two loops, or one loop with a list that goes up and then back down.

for i in range(0, 11):
    led.value = i / 10
    sleep(0.05)
Hint 3: full structure, if you're really stuck 
from gpiozero import PWMLED
from time import sleep

led = PWMLED(17)

while True:
    for i in range(0, 11):
        led.value = i / 10
        sleep(0.05)
    for i in range(10, -1, -1):
        led.value = i / 10
        sleep(0.05)

Push further (optional)

Can you make the fade speed up as the light gets brighter, and slow down as it dims? What line would you need to change?
