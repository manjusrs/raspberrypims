3Your first Python line

Python programs are just text files ending in .py. Let's write one from the terminal.

nano hello.py

Type this inside the editor, then press Ctrl+O to save and Ctrl+X to exit:

print("Hello, Pi!")

Now run it:

python3 hello.py
Expected output

Your terminal should print exactly: Hello, Pi!
